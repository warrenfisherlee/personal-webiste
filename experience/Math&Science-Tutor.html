MCU Programming

With the MCU (such as Arduino or STM32) as the central control unit, the system drives motors and acquires sensor data to enable task-specific mechanism operations. Communication protocols are further implemented to integrate with software, ensuring seamless coordination and efficient system integration.

Control System of 2022 TEL Robot Combat

The system utilizes an STM32 microcontroller to communicate with the software for obtaining target positions. An inverse kinematics algorithm is implemented to calculate the required motor angles, enabling the robotic arm to precisely reach designated locations. Simultaneously, air pumps and solenoid valves are controlled to actuate the end-effector for gripping and releasing objects. In addition, motor velocity data is acquired from the software and combined with encoder feedback, where PI control is applied to ensure accurate distance and speed regulation.

Control System of 2023 Eurobot competition

The system employs an STM32 microcontroller to communicate with the software, obtaining target color sequences and initiating classification. A dedicated classification program optimizes task execution, while photo-interrupters ensure rotational accuracy and pressure sensors monitor the end-effector’s functionality. Additionally, infrared sensors verify task completion and report results back to the software. Throughout the operation, data is recorded in real time to facilitate final task evaluation and performance analysis.

Control System of 2024 Eurobot competition

With the STM32 microcontroller as the central control unit, the system drives a BLDC motor to achieve precise object suction. Task sequences are received through software communication, enabling accurate motor angle control for reliable execution. An infrared distance sensor is also integrated to further enhance positioning accuracy and task robustness.

Control System of 2025 ASME SPDC competition

V1 : The system employs an Arduino Mega to control motor operations, with a synchronized timer that initiates tasks after a 10-second delay. A color sensor module is integrated to classify beads of different materials, directing them to the appropriate channels via a rotary mechanism. To address potential friction between beads and the machine’s inner walls, a vibration motor is incorporated to dislodge stuck beads, ensuring smooth flow and reliable sorting.

V2 : The system employs a vibration motor in combination with a servo motor to regulate the falling speed of beads. A micro switch is used to detect arrival at the target point, upon which the servo motor opens the gate while the vibration motor is simultaneously deactivated, ensuring stable and precise bead delivery.

Control System of 2025 Robotics Dream Camp

The system utilizes a dual ESP32 communication architecture: one ESP32 controls the robot’s chassis and dual-arm movements, while the other is embedded in the remote controller to read joystick and button inputs, transmitting commands and movement directions in real time for precise operation. Additionally, a safety sensor is mounted on the robot’s head to terminate program execution if removed, ensuring safe operation.
